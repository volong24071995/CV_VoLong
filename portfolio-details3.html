Mô tả

Đây là một Dynamic website, thực hiện bởi Teamwork gồm 3 thành viên, với tổng thời gian là 28 ngày.Trong đó bản thân thực hiện 5/9 bảng có quan hệ vơi nhau. Dự án này chúng tôi xây dựng với :

- Ngôn ngữ Java

- Công nghệ Spring Boot

- Front end HTML, CSS, Boostrap, Template Engine Thymeleaf

- Cơ sở dữ liệu là PostgreSQL

- Sử dụng Jpa, Googleapis, @Query để lấy dữ liệu

- Sử dụng Ajax, Jquery, Jquery.dataTables, Summernote, Chart, Validate, Toastr, Bootbox

- Sử dụng Spring HttpSession làm giỏ hàng, checkout sản phẩm

- Sử dụng Spring Security phân quyền truy cập theo ROLE

- Sử dụng ứng dụng IntelliJ

Chức năng

Trang chủ

- Hiển thị danh sách thể loại, sản phẩm mới nhất và bài viết mới nhất

- Tìm kiếm và xem thông tin sản phẩm

- Thêm sản phẩm vào giỏ hàng

- Đăng kí,đăng nhập, thay đổi mật khẩu bằng email tài khoản với role USER

- Checkout giỏ hàng, tăng giảm số lượng, checkout order

Trang admin

- Đăng nhập vào trang quản lý với role ADMIN và EMPLOYEE

- Tạo tài khoản role EMPLOYEE với role ADMIN

- Thêm, sửa, xóa và xem thông tin

- Chart biểu đồ

- Phân trang, tìm kiếm

Kĩ năng thu được

Qua quá trình làm việc nhóm bản thân học được cách phân chia, quản lí công việc bằng Product Backlog và cách phối hợp làm việc giữa các thành viên trong nhóm, tinh thần đồng đội. Biết thêm về công nghệ mới Spring Boot, kết nối với cơ sở dữ liệu mới PostgreSQL để lấy dữ liệu. Biết cách validate dữ liệu backend và frontend, dùng Ajax, Jquery, Jquery.dataTables, Summernote kết hợp HTML, CSS, Boostrap, Template Engine Thymeleaf để vẽ dữ liệu. Sử dụng Spring HttpSession làm giỏ hàng, checkout giỏ hàng và Spring Security để phân quyền. Bản thân biết thêm về thư viện Chart kết hợp Canvas để vẽ biểu đồ thống kê sản phẩm.
